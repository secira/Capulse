Check your entrypoint.sh — it probably runs migrations and then exits without starting the actual web server. The last log shows "Stopping Container" at 11:47:22, which means the process exited.

Fix: Ensure your entrypoint script starts the web server after migrations. It should look something like:

#!/bin/bash
python migrate.py  # or your migration command
