What is a Data API Broker?

A Data API Broker is the broker account we use only for live market data like spot prices, quotes, and option chain. It is separate from your trading account, so you can connect one broker for data and another for orders if needed.

How to connect
  1. Click Data API Broker or Connect Your Broker.
  2. Choose your broker and complete login or OAuth.
  3. Save the connection, then select it as the Data API source in Broker Management.
Broker Connection for Trading

Connecting your broker lets Target Capital place trades on your behalf after you approve an order. We only use the official broker login flow, and you can disconnect anytime from Broker Management.

Holding and Portfolio Access

Once connected, we can read your holdings, positions, and portfolio balance to show live dashboards, risk checks, and account summaries. This helps you track everything in one place without sharing credentials manually.
